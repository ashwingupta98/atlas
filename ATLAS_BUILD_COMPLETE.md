## 📋 To Push to GitHub

### Step 1: Prepare Repository

```bash
cd /app

# Check git status
git status

# If not initialized yet:
git init
git config user.email "dev@atlas-app.com"
git config user.name "Atlas Dev"
```

### Step 2: Add All Files

```bash
git add -A
git status  # Verify files to be committed
```

### Step 3: Create Initial Commit

```bash
git commit -m "feat: Atlas v1.0 - Complete personal finance app

- ✅ 8 pages: Dashboard, Bills, Subscriptions, Tasks, Renewals, Documents, Calendar, Settings
- ✅ 30+ REST API endpoints
- ✅ AI Assistant (Claude Sonnet 4.5)
- ✅ Gmail OAuth + email scanning
- ✅ Cloud document storage
- ✅ Responsive design with organic theme
- ✅ Complete documentation & deployment guides

Tech: React 19, FastAPI, MongoDB, Tailwind CSS, Shadcn/Radix"
```

### Step 4: Add GitHub Remote

```bash
git remote add origin https://github.com/ashwingupta98/atlas.git
```

### Step 5: Push to GitHub

```bash
# If pushing to empty repository:
git branch -M main
git push -u origin main

# If repository already has commits:
git push -u origin main --force-with-lease
```

### Automated Way

```bash
chmod +x push-to-github.sh
./push-to-github.sh
```

## 🎯 Quick Reference

### Start Development

```bash
# 1. Clone
git clone https://github.com/ashwingupta98/atlas.git
cd atlas

# 2. Install
cd backend && pip install -r requirements.txt
cd ../frontend && npm install

# 3. Configure
cp .env.example backend/.env
# Edit backend/.env with MongoDB URL

# 4. Run
# Terminal 1: Backend
cd backend && python -m uvicorn server:app --reload

# Terminal 2: Frontend
cd frontend && npm start

# Access at http://localhost:3000
```
