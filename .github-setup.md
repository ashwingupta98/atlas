# Git Configuration for Atlas

## Setup Instructions

### Initialize Repository Locally

```bash
cd /app

# Initialize git
git init

# Configure user
git config user.email "dev@atlas-app.com"
git config user.name "Atlas Bot"

# Add all files
git add -A

# Create initial commit
git commit -m "feat: Atlas v1.0 - Complete app"

# Add GitHub remote
git remote add origin https://github.com/ashwingupta98/atlas.git

# Push to GitHub
git push -u origin main
```

## GitHub Actions (CI/CD - Optional)

### Frontend Tests
`.github/workflows/frontend-tests.yml`:
```yaml
name: Frontend Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: 18
      - run: npm ci
      - run: npm run build
      - run: npm test -- --coverage
```

### Backend Tests
`.github/workflows/backend-tests.yml`:
```yaml
name: Backend Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-python@v4
        with:
          python-version: 3.11
      - run: pip install -r backend/requirements.txt
      - run: pytest backend/tests
```

## Branching Strategy

```
main (production)
├── develop (staging)
│   ├── feature/bills-improvements
│   ├── feature/ai-enhancements
│   ├── fix/gmail-integration
│   └── ...
└── hotfix/urgent-fix
```

### Creating Feature Branch

```bash
git checkout -b feature/your-feature-name
git add .
git commit -m "feat: describe your feature"
git push origin feature/your-feature-name
# Then create Pull Request on GitHub
```

## Commit Message Convention

```
type(scope): subject

body

footer
```

### Types
- `feat`: New feature
- `fix`: Bug fix
- `docs`: Documentation
- `style`: Code style (formatting, missing semicolons, etc.)
- `refactor`: Code refactoring
- `perf`: Performance improvement
- `test`: Adding tests
- `chore`: Build scripts, dependencies, etc.

### Examples
```
feat(bills): add recurring bill support
fix(gmail): handle OAuth token refresh
docs(setup): add deployment guide
chore(deps): update React to v19.0.1
```

## Useful Commands

```bash
# View commit history
git log --oneline

# View changes before commit
git diff

# Amend last commit
git commit --amend

# View remote URL
git remote -v

# Fetch latest from remote
git fetch origin

# Sync with upstream
git pull origin main

# Stash changes temporarily
git stash

# Apply stashed changes
git stash pop
```

## Releases & Tags

```bash
# Create a release tag
git tag -a v1.0.0 -m "Atlas v1.0.0 - Initial release"
git push origin v1.0.0

# View tags
git tag -l

# Delete local tag
git tag -d v1.0.0

# Delete remote tag
git push origin --delete v1.0.0
```
